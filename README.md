# PCDE-Activity 9.1

## Using GitHub for Version Control

This repository was created as part of **Required GitHub Activity 9.1** in the *Professional Certificate in Data Engineering* program.

The purpose of this activity is to demonstrate basic GitHub workflow including:

- Creating a GitHub repository
- Cloning a repository to a local machine
- Adding files to a repository
- Editing and fixing errors in a Jupyter Notebook
- Committing and pushing changes to GitHub

## Repository Contents

- **MIT.ipynb**  
  A Jupyter Notebook that was edited to remove errors and ensure it runs correctly.
